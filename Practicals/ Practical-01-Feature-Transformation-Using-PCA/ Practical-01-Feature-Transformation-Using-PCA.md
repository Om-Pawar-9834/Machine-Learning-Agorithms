# Practical 01: Feature Transformation Using PCA

## Aim

Apply the **Principal Component Analysis (PCA)** algorithm for dimensionality reduction on a suitable high-dimensional dataset and observe its influence on model performance and computational efficiency.

## Objectives

* Understand the concept of feature transformation.
* Apply PCA to reduce the number of features.
* Compare model performance before and after applying PCA.
* Analyze the effect of dimensionality reduction on training time and prediction accuracy.

## Tools & Libraries Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Dataset

Breast Cancer Wisconsin Dataset (from Scikit-learn)

## Outcome

Successfully applied PCA to reduce the dimensionality of the dataset while preserving most of the important information. Compared the model's accuracy and execution time before and after feature reduction to evaluate the impact of PCA.
